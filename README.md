# odin-recipes
My first project in the Odin Project web course. This is a simple Recipe Website written in HTML with no CSS styling. It is used to practice my skills in basic HTML as well as to get me use to the workflow of working with Git and GitHub.

In this project I will demonstrate skill in basic HTML usage such as boiler plate code, basic elements such as p, h1, body, ul, and so on, and how to make frequent and proper commits to GitHub.
